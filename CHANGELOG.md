# Changelog

## [2.13.0] - 2026-04-16
### Добавлено
- **Compass-Powered Navigation**: Внедрена функция вращения карты по курсу движения (GPS Heading). Карта автоматически разворачивается в сторону движения пользователя.
- **Floating Detail Panels**: Плашки чекпоинтов теперь представлены в виде элегантных парящих карточек с эффектом глубокого размытия.
- **Pull-to-Dismiss Gestures**: Добавлена поддержка нативных мобильных жестов — плашки деталей теперь закрываются плавным свайпом вниз.
- **Smart Marker Compensation**: Маркеры на карте теперь сохраняют читабельность и вертикальную ориентацию при любом угле поворота карты.

### Исправлено
- **Map Interaction Reliability**: Устранены конфликты между CSS-трансформациями и позиционированием Leaflet, восстановлена видимость всех точек.
- **Navigation Controls**: Добавлен новый контрол «Компас» для быстрого переключения режима вращения карты.

## [2.12.1] - 2026-04-15
### Добавлено
- **Sleek HUD 2.1**: Полностью переработанный "стеклянный" интерфейс навигации с золотистыми акцентами.
- **Dynamic Target Badge**: Информация о расстоянии до цели теперь представлена в виде компактного информационного бейджика.
- **Pill-style Header**: Заголовок маршрута переведен в парящую капсулу для визуальной гармонии.
- **High-Contrast Actions**: Нижние кнопки управления получили яркий золотистый градиент для максимальной видимости.

### Исправлено
- **Map Interactivity**: Восстановлена полная управляемость картой во всех режимах навигации (zoom/drag).
- **Layout Harmony**: Исправлены отступы, наложения элементов и визуальная иерархия HUD.
- **Safety Offsets**: Точная настройка отступов от системных меню (top: 64px, bottom: 84px).

## [2.12.0] - 2026-04-14

### Добавлено
- **Новый HUD 2.0**: Полностью переработанный интерфейс навигации с увеличенными шрифтами (22px для статов) и крупным заголовком маршрута.
- **Увеличенная карта**: Высота встроенной карты в деталях маршрута увеличена до 320px для лучшего ориентирования.

### Исправлено
- **Отрисовка карт (Leaflet)**: Исправлена критическая ошибка "белого экрана" на карте. Метод `createTile` переведен в синхронный режим, как того требует движок Leaflet.
- **Стабильность сторов**: Удалены ошибочные зависимости от Pinia. Исправлена реактивность с использованием нативного паттерна проекта (ref/readonly).
- **Маппинг данных**: Исправлена ошибка `TypeError: map is not a function`. Теперь адаптеры корректно обрабатывают координаты (`latitude`/`longitude` vs `lat`/`lng`) и змеиный регистр из Supabase.
- **Watchdog для БД**: Добавлен 3-секундный таймаут для всех операций с IndexedDB (DbService), предотвращающий "зависание" приложения при сбоях локальной базы.
- **Верстка комментариев**: Исправлено отображение имен пользователей и дат в комментариях (приведены к camelCase).
- **Баланс тегов**: Исправлены ошибки компиляции Vite ("missing end tag"), возникшие при рефакторинге шаблона.

### Техническое
- Оптимизирована структура `RouteDetailView.vue`: теперь лишние секции скрываются в активном режиме, отдавая приоритет карте и навигации.
